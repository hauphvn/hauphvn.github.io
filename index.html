Demo Web Api

Đây là cách hiểu và cách làm theo riêng tui á nha. Để phụ cho nhóm kịp Deadline chứ đúng sai thì tui không biết á nha.

Tui đang làm phần web api cho Therapist, nên trong giao diện của therapist gồm những dịch vụ api, mà bên trong những dịch vụ api đó có những phương thức như EditCustomerProfile, CreateCustomer v.v.


Ví dụ khi tui cần Edit thằng customer thì tui click vào button Customer list thì những phương thức trong dịch vụ api lúc nãy tui code nó sẽ thực thi, và mình sẽ dùng những Http request: // GET api/, // POST api/, // PUT api ... để trả lại cho người dùng

Nói chung: Web api là mình thiết kế sẵn những dịch vụ và khi người dùng gọi trên trình duyệt (sử dụng theo giao thức Http, hay nói chung là Http request) thì những dịch vụ này sẽ khởi chạy.


Bây giờ tui sẽ thiết kế Database cơ bản để test, tui làm theo kiểu Database first
